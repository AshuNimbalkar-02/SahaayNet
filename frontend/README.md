# SahaayNet - Frontend 🎨

This is the React + Vite frontend for the SahaayNet platform. It features a modern, responsive "Glassmorphism" design and supports extensive multilingual translations.

## ✨ Key Frontend Features
- **Modern UI**: Built with a custom design system based on translucent "glass" panels.
- **Voice-Enabled Assistant**: Integrated voice interaction for accessibility.
- **Multilingual Support**: Real-time language switching (powered by `translations.js`).
- **Modular Architecture**: Includes independent modules for Blood Aid, Animal Aid, Education, etc.

## 🛠️ Tech Stack
- **Framework**: React 18
- **Build Tool**: Vite
- **Styling**: Vanilla CSS with custom design tokens
- **Icons**: Lucide React

## 🚀 Getting Started
To run the frontend independently:
```bash
npm install
npm run dev
```

For the full project (Frontend + Backend + AI), please refer to the [Main README](../README.md).
